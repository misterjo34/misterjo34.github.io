Projet 1 — Mise en place d’un Active Directory

Système AD / DNS / GPO

Contexte

TP / stage : déploiement d’un domaine pour gérer utilisateurs et postes.

Objectif

  • Créer un domaine (ex : entreprise.local)
  • Structurer OU / groupes / utilisateurs
  • Appliquer des règles via GPO

Technos

  • Windows Server • AD DS • DNS
  • GPO • Partages • NTFS

Résultat

Domaine opérationnel, comptes et politiques appliquées.

Ce que j’ai fait

  1. Installation Windows Server et rôles AD DS/DNS
  2. Promotion en contrôleur de domaine
  3. Création OU (Users/Computers), groupes, utilisateurs
  4. Création GPO (verrouillage, mappage lecteur, restrictions)
  5. Tests : jonction poste au domaine, application des stratégies
